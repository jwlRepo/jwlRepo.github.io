ABE 516x Project

Description
This is the Final project for ABE516x. I will be applying what I've leant in the Fall 2019 sememster on data processing and machine learning to analyze LiDAR point cloud data I have collected.

Project Goal

My hope for this project is to gain more understanding on the relationship between different Tillage settings on a ripper and the resulting soil profile.
I want to see if I can predict tillage setting based on the data collected and alternatively, predict resulting soil profile metric base on tillage setting.

Workflow

Objectives

Find out if I can predict tillage settings base on LiDAR scan.

Investigate What learning models would be best for the LiDAR data I have.

Find out correlation between specific tillage settings and resulting soil measurement.
